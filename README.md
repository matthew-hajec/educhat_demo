# EduChat Demo

## Production Environment Variables

### 1. `OPENAI_TOKEN`

- **Description**: The API token used to authenticate requests to the OpenAI API.
- **Example**: `sk-abcdefghijklmnopqrstuvwxyz1234567890`

### 2. `OPENAI_ASSISTANT_ID`

- **Description**: The ID of the OpenAI assistant or model you are interacting with.
- **Example**: `assistant-1234abcd5678efgh`

### 3. `NODE_ENV`

- **Description**: Specifies the Node.js environment mode. Should be set to `production` in a production environment for optimized performance.
- **Example**: `production`

### 4. `PORT`

- **Description**: The port number on which the server will listen for incoming connections.
- **Example**: `3000`
